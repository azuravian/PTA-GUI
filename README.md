# Plex Themesong Downloader
Assists with downloading theme songs for your movies from either the Plex Themes Google Drive, or from Youtube if not found there.


Installation Instructions:
1. Install Python 3.9
2. Install pipenv using "pip install pipenv".
3. Download and unzip this repository.
4. In the folder where your "PlexThemeAutoGui.py" files are, run "pipenv install"


To run:
    pipenv run python PlexThemeAutoGui.py
